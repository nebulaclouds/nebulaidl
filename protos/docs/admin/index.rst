Nebula Admin Service entities
============================

These are the control plane entities that can be used to communicate with the
NebulaAdmin service over gRPC or REST. The endpoint specification is defined in the
`Admin raw protos <https://github.com/nebulaclouds/nebulaidl/blob/master/protos/nebulaidl/admin>`__
